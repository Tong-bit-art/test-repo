# 人员库管理系统权限测试用例

## 测试环境
- 后端：Spring Boot + Spring Security
- 前端：Vue3 + Vue Router
- 数据库：MySQL
- 测试账号：
  - 管理员：admin / 123456
  - 录入员：entry / 123456
