# 人员库管理系统部署说明

## 1. 环境要求

- Java 17 或以上
- Maven 3.6 或以上
- Node.js 16 或以上
- MySQL 8.0 或以上

## 2. 数据库初始化

1. 启动 MySQL 服务
2. 执行以下步骤创建数据库和表结构：

   ```bash
   # 登录 MySQL
   mysql -u root -p
   
   # 执行建表脚本
   source d:\test-repo\personnel-management-system\backend\src\main\resources\schema.sql
   ```

3. 数据库配置：
   - 数据库名：`personnel_system`
   - 用户名：`root`
   - 密码：`123456`

   如果需要修改数据库配置，请编辑 `backend/src/main/resources/application.yml` 文件。

## 3. 后端服务启动

1. 进入后端目录：
   ```bash
   cd d:\test-repo\personnel-management-system\backend
   ```

2. 构建项目：
   ```bash
   mvn clean package
   ```

3. 启动应用：
   ```bash
   java -jar target/system-1.0.0.jar
   ```

   后端服务将在 `http://localhost:8080` 上运行。

## 4. 前端服务启动

1. 进入前端目录：
   ```bash
   cd d:\test-repo\personnel-management-system\frontend
   ```

2. 安装依赖：
   ```bash
   npm install
   ```

3. 启动开发服务器：
   ```bash
   npm run dev
   ```

   前端服务将在 `http://localhost:3000` 上运行。

## 5. 登录系统

打开浏览器访问 `http://localhost:3000`，使用以下账号登录：

- 用户名：`admin`
- 密码：`123456`

### 录入员账号：
- 用户名：`entry`
- 密码：`123456`

## 6. 功能说明

### 超级管理员权限：
- 人员管理：查看、筛选、删除人员信息，导出Excel
- 随机抽取：纯随机抽取、指定范围抽取、指定人员抽取
- 专业类目管理：增删改查专业类目
- 用户管理：增删改查子账号

### 录入员权限：
- 人员录入：手动录入人员信息，Excel批量导入

## 7. 注意事项

1. 确保 MySQL 服务正常运行
2. 确保后端服务和前端服务都已启动
3. 首次登录时使用默认账号密码
4. 生产环境中请修改默认密码
5. 导入Excel时请确保文件格式正确
